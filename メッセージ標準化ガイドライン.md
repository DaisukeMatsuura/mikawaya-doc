## 2. エラーメッセージ

### 2.1 バリデーションエラー（入力エラー）

#### メールアドレス関連

| 項目 | エラーメッセージ |
| --- | --- |
| 必須チェック | 「メールアドレスを入力してください。」 |
| 形式チェック | 「正しいメールアドレスを入力してください。」 |
| 重複チェック | 「このメールアドレスは既に登録されています。」 |

#### パスワード関連

| 項目 | エラーメッセージ |
| --- | --- |
| 必須チェック | 「パスワードを入力してください。」 |
| 最小文字数 | 「パスワードは8文字以上で入力してください。」 |
| 不一致 | 「パスワードが一致しません。」 |
| 強度不足 | 「パスワードは大文字・小文字・数字を含めてください。」 |

#### 汎用フィールド

| 項目 | エラーメッセージ |
| --- | --- |
| 必須項目未入力 | 「○○を入力してください。」 |
| 形式エラー | 「○○の形式が正しくありません。」 |
| 文字数超過 | 「○○は○○文字以内で入力してください。」 |
| 数値形式エラー | 「○○は数値で入力してください。」 |

### 2.2 認証・認可エラー

| 状況 | エラーメッセージ |
| --- | --- |
| ログイン失敗（ID/PW不一致） | 「メールアドレスまたはパスワードが正しくありません。」 |
| アカウントロック（5回失敗） | 「アカウントがロックされています。30分後に再度お試しいただくか、パスワード再設定を行ってください。」 |
| セッション有効期限切れ | 「セッションの有効期限が切れました。再度ログインしてください。」 |
| アクセス権限なし | 「このページへのアクセス権限がありません。」 |
| 削除されたアカウント | 「このアカウントは無効です。管理者にお問い合わせください。」 |

### 2.3 APIエラー

| HTTP ステータス | メッセージ |
| --- | --- |
| 400 Bad Request | 「リクエストが無効です。入力内容を確認してください。」 |
| 401 Unauthorized | 「認証が必要です。ログインしてください。」 |
| 403 Forbidden | 「このリソースへのアクセス権限がありません。」 |
| 404 Not Found | 「リソースが見つかりません。」 |
| 409 Conflict | 「リソースが重複しています。」 |
| 429 Too Many Requests | 「リクエストが多すぎます。少し待ってから試してください。」 |
| 500 Internal Server Error | 「サーバーエラーが発生しました。管理者にお問い合わせください。」 |
| 503 Service Unavailable | 「サービスが一時的に利用できません。しばらく後でお試しください。」 |

### 2.4 エラーメッセージの表示

| 項目 | 仕様 |
| --- | --- |
| 入力フィールドエラー | 該当フィールドを赤枠で囲む + フィールド直下にメッセージ表示 |
| 全体エラー（トースト） | 画面右下にトースト通知 |
| APIエラー（トースト） | 画面右下にトースト通知（3秒表示） |
| ダイアログエラー | モーダル内にメッセージ表示 |
| アイコン | エラー: ✕ または ⚠️（赤） |
| 背景色 | エラー: 薄赤 |

## 5. メッセージの実装ガイドライン

### 5.1 ユーザーフレンドリーなメッセージの書き方

✅ **推奨される書き方**:
- 敬語を使う（ですます調）
- 短く簡潔に（1文）
- 具体的に何が起こったかを説明
- 次のアクション（ある場合）を示唆

❌ **避けるべき書き方**:
- 技術的な用語や暗号的なメッセージ
- 複数行のメッセージ
- 曖昧な表現

### 5.2 例

**悪い例**:
```
"Exception: NullPointerException in UserValidator"
"Request timeout error occurred at 12:34:56"
```

**良い例**:
```
"メールアドレスを入力してください。"
"サーバーとの通信がタイムアウトしました。接続を確認してしばらく待ってからお試しください。"
```

## 6. メッセージ標準化チェックリスト（開発時確認）

### 成功メッセージ
- [ ] 全てのCRUD操作で成功メッセージを表示
- [ ] メッセージ形式が「○○を××しました」に統一
- [ ] トースト通知は画面右下に3秒表示

### エラーメッセージ
- [ ] 必須項目チェック: 「○○を入力してください。」
- [ ] 形式チェック: 「正しい○○を入力してください。」
- [ ] ログイン失敗: 「メールアドレスまたはパスワードが正しくありません。」
- [ ] アクセス権限: 「このページへのアクセス権限がありません。」

### データなしメッセージ
- [ ] 一覧画面の空状態: 「○○が登録されていません。」に統一
- [ ] 検索結果なし: 「該当する○○が見つかりません。」に統一

### 確認ダイアログ
- [ ] 削除操作: 「この操作は取り消せません。」を明記
- [ ] 再送信: 対象ユーザーを明記
