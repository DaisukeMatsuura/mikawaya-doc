# コンシェルジュ 案件情報更新（モーダル）

## Figmaデザイン

※案件詳細画面から呼び出される編集モーダル

---

## 🎯 画面の目的

コンシェルジュが担当案件の訪問予定や作業メモを更新する。新規案件の作成はできない（パートナー/オペレータが行う）。

---

## 📊 表示データ項目

### 案件情報（表示のみ）

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 案件No | String | - | 表示のみ |
| お客様名 | String | - | 表示のみ |
| 担当店舗 | String | - | 表示のみ |
| サポート種別 | String | - | 表示のみ |
| サービス内容 | String | - | 表示のみ |

### 訪問情報（編集可能）

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 訪問予定日 | Date | - | 日付選択 |
| 訪問予定時間 | Time | - | セレクトまたは入力 |
| 訪問までの対応履歴 | String | - | テキストエリア |

### 作業メモ（編集可能）

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| コンシェルジュメモ | String | - | テキストエリア、自分専用メモ |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 訪問予定日 | 選択 | カレンダーから日付選択 |
| 訪問予定時間 | 選択/入力 | 時間を選択または入力 |
| 対応履歴テキストエリア | 入力 | 自由入力（追記形式推奨） |
| コンシェルジュメモ | 入力 | 自由入力（自分のみ閲覧可能） |
| キャンセルボタン | クリック | モーダルを閉じる（入力内容は破棄） |
| 保存ボタン | クリック | 変更を保存 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 編集モード | 既存データを表示 |
| バリデーションエラー | 該当項目を赤枠 + エラーメッセージ |
| 処理中 | ボタンをローディング表示 |
| 保存完了 | モーダルを閉じる、詳細画面を更新、トースト通知「案件情報を更新しました」 |

---

## 🔐 権限

| ユーザー種別 | 新規作成 | 編集 |
| --- | --- | --- |
| 管理者 | ◯ | ◯ |
| オペレータ | ◯ | ◯ |
| パートナー | ◯ | ◯ |
| コンシェルジュ | ✕ | △（担当案件のみ、限定項目） |

※ コンシェルジュは新規案件作成不可、担当案件の限定項目のみ編集可能

---

## 📝 備考

- API: `PATCH /api/concierge/cases/{id}`
- コンシェルジュは新規案件を作成できない
- 編集できるのは担当案件のみ
- 編集可能な項目は限定的（訪問予定、対応履歴、メモのみ）
- お客様情報、サービス内容、受注可否などの変更はパートナー/オペレータに依頼
- コンシェルジュメモは本人のみ閲覧可能（店舗・オペレータには非表示）
- 変更履歴は監査ログに記録

---

## 編集可能項目の詳細

| 項目 | 編集可否 | 理由 |
| --- | --- | --- |
| 訪問予定日 | ◯ | 顧客との調整で変更の可能性あり |
| 訪問予定時間 | ◯ | 顧客との調整で変更の可能性あり |
| 訪問までの対応履歴 | ◯ | 進捗を記録するため |
| コンシェルジュメモ | ◯ | 個人的な作業メモ |
| 案件No | ✕ | システム管理項目 |
| お客様名 | ✕ | マスタ変更が必要 |
| 担当店舗 | ✕ | 管理者権限が必要 |
| サポート種別 | ✕ | パートナーが設定 |
| サービス内容 | ✕ | パートナーが設定 |
| 受注可否 | ✕ | パートナーが管理 |
| 見積もり金額 | ✕ | パートナーが設定 |
