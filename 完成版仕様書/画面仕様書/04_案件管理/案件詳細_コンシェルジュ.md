## 🎯 画面の目的

コンシェルジュが担当案件の詳細情報を確認し、編集やコピーを行う。

---

## 📊 表示データ項目

### 基本情報

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 案件No | String |  |
| ステータス | Enum | バッジ表示 |
| 顧客（住所） | String |  |
| 担当店舗 | String | 鉛筆アイコンで編集可 |
| 顧客（電話番号） | String |  |
| お客様名 | String | 鉛筆アイコンで編集可 |
| 顧客（携帯番号） | String |  |
| サポート種別 | String |  |

### コールセンターからの取次事項

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 受付日 | DateTime |  |
| 作成日からの経過日 | Number | 自動計算 |
| 取次内容 | String | 長文テキスト、ツールチップあり |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 編集ボタン | タップ | 案件編集画面へ遷移 |
| 案件コピーボタン | タップ | 案件作成画面（内容コピー）へ遷移 |
| 担当店舗の鉛筆アイコン | タップ | 該当項目の編集モーダルを表示 |
| お客様名の鉛筆アイコン | タップ | 該当項目の編集モーダルを表示 |
| パンくずリスト | タップ | 各階層へ遷移 |

## 🔐 権限

| ユーザー種別 | 閲覧 | 編集 |
| --- | --- | --- |
| 管理者 | ✕ | ✕ |
| オペレータ | ✕ | ✕ |
| パートナー | ✕ | ✕ |
| コンシェルジュ | ◯ | ◯ |

※ 自分が担当する案件のみ閲覧・編集可能

---

## 📝 備考

- API: `GET /api/concierge/cases/{id}`
- スマホ専用画面（レスポンシブ対応）
- 自分が担当する案件のみ閲覧可能
- オペレーター/パートナー版と比べて表示項目は簡略化（コンシェルジュに必要な情報のみ）
- 一部項目は鉛筆アイコンからインライン編集可能
- 「案件コピー」でリピート案件を素早く作成可能

---

## オペレーター/パートナー版との違い

| 項目 | オペレーター/パートナー | コンシェルジュ |
| --- | --- | --- |
| 表示項目 | 全項目（売上情報、報酬等含む） | 基本情報 + 取次事項のみ |
| 削除機能 | あり | なし |
| 対応デバイス | PC | スマホ |
