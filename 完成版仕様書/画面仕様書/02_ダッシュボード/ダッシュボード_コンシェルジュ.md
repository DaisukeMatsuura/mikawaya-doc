## 🎯 画面の目的

コンシェルジュが自分の担当案件の状況を一目で把握し、訪問予定を確認できるようにする。

---

## 📊 表示データ項目

### サマリーカード

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 訪問予定 | Number | 件数表示 |
| 訪問日が未決定 | Number | 件数表示 |
| 完了報告-未報告 | Number | 件数表示 |

### 月間実績

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| サポート件数 | Number | 当月の件数 |
| 売上金額 | Number | 当月の売上（円） |

### 次の訪問予定

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 訪問日 | Date | 「本日」バッジ表示あり |
| 訪問時間 | Time |  |
| 案件No | String |  |
| お客様名 | String |  |
| ステータス | Enum | バッジ表示（訪問予定） |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| ハンバーガーメニュー | タップ | サイドメニューを表示 |
| 訪問予定カード | タップ | 訪問予定一覧へ遷移 |
| 訪問日が未決定カード | タップ | 訪問日未決定の案件一覧へ遷移 |
| 完了報告-未報告カード | タップ | 未報告案件一覧へ遷移 |
| 次の訪問予定の行 | タップ | 案件詳細画面へ遷移 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | サマリー + 月間実績 + 次の訪問予定を表示 |
| 訪問予定なし | 「本日の訪問予定はありません。」 |

## 📝 備考

- API: `GET /api/concierge/dashboard`
- スマホ専用画面（レスポンシブ対応）
- 自分が担当する案件のみ表示
- 月間実績は当月（表示月）のデータを集計
- 月間実績の「売上金額」は報酬金額を表示
- 「本日」バッジは訪問日が当日の場合に表示
- 次の訪問予定は直近の予定を最大5件表示
