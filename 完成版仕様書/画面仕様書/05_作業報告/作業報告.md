## 🎯 画面の目的

コンシェルジュがサポート完了後に作業内容・売上情報を報告する。

---

## 📊 表示データ項目

### ヘッダー情報（表示のみ）

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 案件番号 | String |  |
| お客様名 | String |  |
| 訪問日時 | DateTime |  |
| 報告者 | String | コンシェルジュ名 |

### 会話内容・サポート記録画像

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 会話内容 | String | ◯ | テキストエリア |
| サポート記録画像 | File | - | 画像アップロード（SVG, PNG, JPG, GIF / max 3MB）、複数可 |

### サポート内容詳細

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| サポート時間 | Enum | ◯ | セレクト |
| 代金回収方法 | Enum | ◯ | セレクト |
| サポート料金総額（税込） | Number | ◯ | お客様への請求金額 |
| 交通費（税込） | Number | - |  |
| 無料券（税込） | Number | - | ※無料券を使用した場合にその金額を記入 |
| 経費・外注費（税込） | Number | - | ※サポートを行うためにかかった外注費・経費を記入 |
| 経費・原価費 | Number | - | ※サポートを完了するために使用した外注・経費の詳細を記入 |
| サポート種別 | Enum | ◯ | セレクト |
| 商品カテゴリ | Enum | ◯ | セレクト（複数選択） |
| お客様にサポートの案内などをしましたか？ | Enum | - | セレクト ※お客様が今まで利用していないサービスや新サービスの案内を行ったか |
| 依頼のきっかけを選択 | Enum | - | セレクト ※お客様が今回のサポートを依頼したきっかけ。わからない場合は「その他」 |
| サポートメンバー① | Enum | - | セレクト（コンシェルジュ選択）※一緒にサポートを手伝ってくれたコンシェルジュ |
| サポートメンバー② | Enum | - | セレクト（コンシェルジュ選択） |

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 会話内容テキストエリア | 入力 | 必須バリデーション |
| 画像アップロードエリア | タップ | ファイル選択ダイアログを表示 |
| アップロード済みファイル | タップ | プレビュー表示 or 削除 |
| サポート時間セレクト | タップ | 時間を選択 |
| 代金回収方法セレクト | タップ | 回収方法を選択 |
| 金額入力欄 | 入力 | 数値のみ許可 |
| サポート種別セレクト | タップ | 種別を選択 |
| 商品カテゴリセレクト | タップ | カテゴリを複数選択 |
| サポートメンバー追加ボタン | タップ | サポートメンバー選択欄を追加 |
| キャンセルボタン | タップ | 画面を閉じる（入力内容は破棄） |
| 下書き保存ボタン | タップ | 下書きとして保存 |
| 保存ボタン | タップ | 報告を完了 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ヘッダー情報を表示、入力欄は空欄 |
| 下書き編集 | 下書き保存した内容を表示 |
| 画像アップロード中 | ローディング表示 |
| バリデーションエラー | 該当項目を赤枠 + エラーメッセージ |
| 処理中 | ボタンをローディング表示 |
| 保存完了 | 一覧へ戻る、トースト通知「作業報告を完了しました」 |
| 下書き保存完了 | トースト通知「下書きを保存しました」 |

## 📝 備考

- API: `POST /api/concierge/cases/{id}/report`（報告完了）
- API: `POST /api/concierge/cases/{id}/report/draft`（下書き保存）
- スマホ専用画面（レスポンシブ対応）
- 画像は複数アップロード可能（Before/After等の記録用）
- サポートメンバーは「+サポートメンバーを追加」で動的に追加可能
- 下書き保存で途中保存可能
- 報告完了後、案件ステータスが「報告完了」に更新される
