## 🎯 画面の目的

システム利用者（オペレーター、パートナー加盟店、コンシェルジュ）のアカウントを一覧表示し、管理する。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| アイコン | String | - | 名前のイニシャル（例：CN） |
| 名前 | String | ◯ | 姓 + 名 |
| メールアドレス | String | ◯ | ソート可能 |
| アカウント種別 | Enum | ◯ | オペレーター / パートナー加盟店 / コンシェルジュ |
| アクティブ状況 | Enum | ◯ | アクティブ / 非アクティブ |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 名前検索 | 入力 | 名前で絞り込み（部分一致） |
| 名前カラムヘッダー | クリック | 名前で昇順/降順ソート |
| メールアドレスカラムヘッダー | クリック | メールアドレスで昇順/降順ソート |
| アクティブ状況セレクト | 変更 | その場でステータス更新 |
| 三点メニュー（︙） | クリック | メニュー表示（編集、削除等） |
| 新規アカウント発行ボタン | クリック | アカウント発行モーダルを表示 |
| ページネーション | クリック | ページ切り替え |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | 一覧表示（1ページ10件） |
| データなし | 「アカウントが登録されていません。」 |
| 検索結果なし | 「該当するアカウントが見つかりません。」 |
| ステータス変更中 | セレクトをローディング表示 |
| ステータス変更完了 | トースト通知「ステータスを変更しました」 |

---

## 🔐 権限

| ユーザー種別 | 閲覧 | 作成 | 編集 | 削除 |
| --- | --- | --- | --- | --- |
| 管理者 | ◯ | ◯ | ◯ | ◯ |
| オペレータ | ✕ | ✕ | ✕ | ✕ |
| パートナー | ✕ | ✕ | ✕ | ✕ |
| コンシェルジュ | ✕ | ✕ | ✕ | ✕ |

---

## 📝 備考

- API: `GET /api/admin/accounts`
- サイドメニューでアカウント種別ごとに絞り込み可能（オペレーター / パートナー加盟店 / コンシェルジュ）
- アクティブ状況を「非アクティブ」にするとログイン不可になる
- 削除は物理削除ではなく論理削除（非アクティブ化）を推奨
- **自分自身のアカウントは三点メニュー非表示（編集・削除不可）**

---

## 三点メニュー

| 項目 | 動作 |
| --- | --- |
| パスワードリセット | パスワードリセットメールを送信 |
| 編集 | アカウント編集モーダルを表示 |
| 削除 | 削除確認ダイアログを表示 |

---

## ダイアログ

### パスワードリセット確認

| 項目 | 内容 |
| --- | --- |
| タイトル | パスワードリセット |
| メッセージ | 「○○さんにパスワードリセットメールを送信します。よろしいですか？」 |
| ボタン | キャンセル / 送信 |

### 削除確認

| 項目 | 内容 |
| --- | --- |
| タイトル | アカウント削除 |
| メッセージ | 「○○さんのアカウントを削除します。この操作は取り消せません。よろしいですか？」 |
| ボタン | キャンセル / 削除 |
