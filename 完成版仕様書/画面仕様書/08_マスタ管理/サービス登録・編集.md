## 🎯 画面の目的

サービスの新規登録および既存サービスの編集を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| サービス名 | String | ◯ |  |
| サービス名（ふりがな） | String | ◯ | ひらがなのみ |
| 基本料金（税抜） | Number | ◯ | 数値のみ |
| 有効/無効 | Enum | ◯ | 有効 / 無効、デフォルト：有効 |
| サポート種別 | Enum | ◯ | セレクト（簡単サポート / プロサポート） |
| 商品カテゴリ | Enum | ◯ | セレクト（マスタから選択） |
| 説明 | String | - | テキストエリア |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| サービス名入力欄 | 入力 | 必須バリデーション |
| サービス名（ふりがな）入力欄 | 入力 | ひらがなのみ許可 |
| 基本料金入力欄 | 入力 | 数値のみ許可 |
| 有効/無効セレクト | 選択 | 有効 / 無効 |
| サポート種別セレクト | 選択 | マスタから選択 |
| 商品カテゴリセレクト | 選択 | マスタから選択 |
| 説明テキストエリア | 入力 | 任意入力 |
| キャンセルボタン | クリック | モーダルを閉じる（入力内容は破棄） |
| 登録ボタン | クリック | 新規登録処理を実行 |
| 更新ボタン（編集時） | クリック | 更新処理を実行 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 新規登録モード | フォーム空欄、ボタン「登録」 |
| 編集モード | 既存データを表示、ボタン「更新」 |
| バリデーションエラー | 該当項目を赤枠 + エラーメッセージ |
| 処理中 | ボタンをローディング表示 |
| 登録/更新完了 | モーダルを閉じる、一覧を更新、トースト通知 |

## 📝 備考

- API: `POST /api/admin/services`（新規登録）
- API: `PUT /api/admin/services/{id}`（編集）
- サポート種別・商品カテゴリはマスタ管理で登録されたものから選択
- 新規登録時のデフォルトは「有効」
