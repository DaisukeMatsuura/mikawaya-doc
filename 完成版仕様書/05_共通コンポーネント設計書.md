## 共通UIコンポーネント

### 1. ボタン（Button）

#### バリアント

| バリアント | 用途 | 背景 | テキスト |
|-----------|------|------|---------|
| primary | 主要アクション | Primary-500 | White |
| secondary | 副次アクション | Secondary-100 | Secondary-700 |
| outline | 軽いアクション | Transparent | Primary-500 |
| danger | 削除・危険操作 | Error-500 | White |
| ghost | テキストリンク風 | Transparent | Primary-500 |

#### サイズ

| サイズ | 高さ | パディング | フォントサイズ |
|-------|------|-----------|--------------|
| sm | 32px | 12px 16px | 14px |
| md | 40px | 12px 20px | 16px |
| lg | 48px | 16px 24px | 18px |

#### 状態

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  ボタン状態                                                                      │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐         │
│  │  通常    │  │ ホバー   │  │ アクティブ│  │ フォーカス│  │ 無効    │         │
│  │          │  │          │  │          │  │          │  │          │         │
│  │ Primary  │  │ Primary  │  │ Primary  │  │ Primary  │  │ Gray     │         │
│  │   500    │  │   600    │  │   700    │  │ 500+Ring │  │   300    │         │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘  └──────────┘         │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

#### コード例

```tsx
// Button コンポーネント
interface ButtonProps {
  variant?: 'primary' | 'secondary' | 'outline' | 'danger' | 'ghost';
  size?: 'sm' | 'md' | 'lg';
  disabled?: boolean;
  loading?: boolean;
  icon?: React.ReactNode;
  children: React.ReactNode;
  onClick?: () => void;
}

<Button variant="primary" size="md">
  保存
</Button>

<Button variant="danger" size="sm" icon={<TrashIcon />}>
  削除
</Button>
```

### 5. トースト通知（Toast）

#### タイプ

| タイプ | アイコン | 背景色 | 用途 |
|-------|---------|-------|------|
| success | CheckCircle | Success-100 | 操作成功 |
| error | XCircle | Error-100 | エラー発生 |
| warning | AlertTriangle | Warning-100 | 警告 |
| info | InfoCircle | Info-100 | 情報 |

#### 表示位置

- PC: 右上（top-right）
- スマホ: 上部中央（top-center）

#### 動作

| 項目 | 値 |
|------|-----|
| 表示時間 | 5秒（success/info）、10秒（error/warning） |
| アニメーション | スライドイン（右から） |
| 複数表示 | 最大3件、スタック表示 |
| 手動閉じ | ×ボタンクリック |
