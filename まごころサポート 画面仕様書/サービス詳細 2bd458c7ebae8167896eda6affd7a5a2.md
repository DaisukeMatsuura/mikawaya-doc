## 🎯 画面の目的

サービスの詳細情報を表示し、編集・削除を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| サービスカテゴリ名 | String | ◯ |  |
| 基本料金（税抜） | Number | ◯ |  |
| 有効/無効 | Enum | ◯ | 有効 / 無効 |
| 説明 | String | - | 全文表示 |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 無効にするボタン | クリック | 無効確認ダイアログを表示 |
| 編集ボタン（右上） | クリック | サービス編集モーダルを表示 |

※無効のサービスには「有効にする」ボタンを表示

---

## ダイアログ

### 無効確認

| 項目 | 内容 |
| --- | --- |
| タイトル | 無効にする |
| メッセージ | 「○○を無効にします。案件作成時に選択できなくなります。よろしいですか？」 |
| ボタン | キャンセル / 無効にする |

### 有効確認

| 項目 | 内容 |
| --- | --- |
| タイトル | 有効にする |
| メッセージ | 「○○を有効にします。よろしいですか？」 |
| ボタン | キャンセル / 有効にする |

---

## 📝 備考

- API: `GET /api/admin/services/{id}`
- API: `PATCH /api/admin/services/{id}`（ステータス変更）
- 削除は論理削除（ステータスを「無効」に変更）
- 無効のサービスは案件作成時に選択不可
- 過去の案件データとの紐づきは維持される
