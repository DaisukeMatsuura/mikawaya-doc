## 🎯 画面の目的

お客様（顧客）の一覧を表示し、顧客情報の管理を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 店舗 | String | ◯ |  |
| お客様名/取引先 | String | ◯ |  |
| 住所 | String | ◯ |  |
| 利用回数 | Number | ◯ |  |
| 前回利用の日付 | Date | - |  |
| 前回のサポート内容 | String | - |  |
| 担当コンシェルジュ | String | - |  |
| 性別 | Enum | - | 男 / 女 |
| 年齢 | Number | - |  |
| 備考 | String | - |  |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| お客様検索 | 入力 | 電話番号/名前/住所で絞り込み（部分一致） |
| 行 | クリック | お客様詳細画面へ遷移 |
| 三点メニュー（︙） | クリック | メニュー表示 |
| 新規お客様登録ボタン | クリック | お客様登録モーダルを表示 |
| ページネーション | クリック | ページ切り替え |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | 一覧表示（1ページ10件） |
| データなし | 「お客様情報が登録されていません。」 |
| 検索結果なし | 「該当するお客様が見つかりません。」 |

---

## 🔐 権限

| ユーザー種別 | 閲覧 | 作成 | 編集 |
| --- | --- | --- | --- |
| 管理者 | ◯（全件） | ◯ | ◯ |
| オペレータ | ◯（全件） | ◯ | ◯ |
| パートナー | △（自店舗のみ） | ◯ | ◯ |
| コンシェルジュ | △（自店舗のみ） | ◯ | ◯ |

※ パートナー・コンシェルジュは自分の所属する店舗に関連するお客様のみ閲覧可能

---

## 三点メニュー

| 項目 | 動作 |
| --- | --- |
| 編集 | お客様編集モーダルを表示 |
| このお客様のサポート依頼 | 案件作成モーダルを表示（お客様がプリセット） |

---

## 📝 備考

- API: `GET /api/operator/customers`
- 検索は電話番号、名前、住所で部分一致検索
- 「このお客様のサポート依頼」からリピート案件を素早く作成可能
- 利用回数・前回利用日・前回サポート内容は案件データから自動集計
