## 🎯 画面の目的

パートナー加盟店の担当者が自店舗の案件状況を一目で把握し、対応が必要な案件を迅速に処理できるようにする。

---

## 📊 表示データ項目

### サマリーカード

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 確認待ち案件 | Number | コンシェルジュ割当待ちの件数 |
| 進行中案件 | Number | 対応中の案件数 |
| 完了報告待ち | Number | コンシェルジュの作業報告待ち件数 |

### 月間実績

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 対応件数 | Number | 当月の案件数 |
| 売上金額 | Number | 当月の売上（円） |

### コンシェルジュ一覧（簡易）

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| コンシェルジュ名 | String |  |
| 担当件数 | Number | 現在の担当案件数 |
| 稼働状況 | Enum | アクティブ/非アクティブ |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| ハンバーガーメニュー | クリック | サイドメニューを表示 |
| 確認待ち案件カード | クリック | 確認待ち案件一覧へ遷移 |
| 進行中案件カード | クリック | 進行中案件一覧へ遷移 |
| 完了報告待ちカード | クリック | 完了報告待ち一覧へ遷移 |
| コンシェルジュの行 | クリック | コンシェルジュ詳細画面へ遷移 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | サマリー + 月間実績 + コンシェルジュ一覧を表示 |
| データなし | 「対応する案件はありません。」 |

| --- | --- |

## 📝 備考

- API: `GET /api/partner/dashboard`
- PC対応（Responsive Web Design）
- パートナー加盟店の自店舗情報のみ表示
- 月間売上は当月（表示月）のデータを集計
- コンシェルジュ一覧は最大5名表示
- 売上金額は報酬金額を表示
