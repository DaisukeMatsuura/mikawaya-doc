## 🎯 画面の目的

提供サービスの一覧を表示し、サービスの管理を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| サービス名 | String | ◯ |  |
| 基本料金 | Number | ◯ | 税抜表示 |
| サポート種別 | Enum | ◯ | 簡単サポート / プロサポート 等 |
| 商品カテゴリ | String | ◯ | 買い物/掃除/草むしり 等 |
| 有効/無効 | Enum | ◯ | 有効 / 無効 |
| 説明 | String | - | 省略表示（詳細で全文表示） |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| サービス名で検索 | 入力 | サービス名で絞り込み（部分一致） |
| 商品カテゴリセレクト | 選択 | カテゴリで絞り込み |
| 行 | クリック | サービス詳細画面へ遷移 |
| 三点メニュー（︙） | クリック | メニュー表示（編集、詳細） |
| サービス登録ボタン | クリック | サービス登録モーダルを表示 |
| ページネーション | クリック | ページ切り替え |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | 一覧表示（1ページ10件） |
| データなし | 「サービスが登録されていません。」 |
| 検索結果なし | 「該当するサービスが見つかりません。」 |

## 三点メニュー

| 項目 | 動作 |
| --- | --- |
| 編集 | サービス編集モーダルを表示 |
| 詳細 | サービス詳細画面へ遷移 |

---

## 📝 備考

- API: `GET /api/admin/services`
- 有効/無効は一覧上で視覚的に区別（有効：緑バッジ、無効：バッジなし）
- 無効のサービスは案件作成時に選択不可
- 削除は物理削除ではなく「無効」にする運用（論理削除）
