# コンシェルジュ詳細

ステータス: 確定
ユーザー種別: パートナー加盟店
概要: コンシェルジュの詳細表示と統計
機能番号: M-003
画面種別: 詳細

---

## 🎯 画面の目的

パートナー加盟店の担当者が、特定のコンシェルジュの詳細情報（個人情報、担当案件、実績）を確認できるようにする。

---

## 📊 表示データ項目

### 個人情報セクション

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| コンシェルジュ名 | String | 氏名 |
| ふりがな | String | ひらがな |
| 電話番号 | String | 携帯またはIP電話 |
| メールアドレス | Email | 連絡先メール |
| 生年月日 | Date | 年齢計算用 |
| 住所 | String | 住所地 |

### 勤務情報セクション

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 雇用形態 | Enum | 正社員/契約社員/パート/アルバイト |
| 入社日 | Date | 勤務開始日 |
| ステータス | Enum | アクティブ / 非アクティブ |

### 統計情報セクション

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| 累計対応件数 | Number | 全期間 |
| 月間対応件数 | Number | 当月 |
| 月間売上 | Number | 当月の報酬金額（円） |
| 平均評価 | Decimal | 5段階（ある場合） |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 戻るボタン | クリック | コンシェルジュ一覧へ遷移 |
| 編集ボタン | クリック | コンシェルジュ登録・編集モーダルを表示 |
| 削除ボタン | クリック | 削除確認ダイアログを表示 |
| 担当案件タブ | クリック | 担当中の案件一覧を表示 |
| 実績グラフ | ホバー | 詳細情報をツールチップで表示 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | 個人情報 + 勤務情報 + 統計情報を表示 |
| データなし | 「コンシェルジュが見つかりません。」メッセージ |

---

## 🔐 権限

| ユーザー種別 | 閲覧 | 編集 | 削除 |
| --- | --- | --- | --- |
| 管理者 | ✕ | ✕ | ✕ |
| オペレータ | ✕ | ✕ | ✕ |
| パートナー | ◯（自店舗） | ◯ | ◯ |
| コンシェルジュ | ✕ | ✕ | ✕ |

---

## 🗑️ 削除ダイアログ

| 項目 | 内容 |
| --- | --- |
| タイトル | コンシェルジュ削除 |
| メッセージ | 「○○（コンシェルジュ名）を削除します。この操作は取り消せません。よろしいですか？」 |
| ボタン | キャンセル / 削除 |

---

## 📝 備考

- API: `GET /api/partner/concierges/{id}`
- PC対応（Responsive Web Design）
- パートナー加盟店は自店舗のコンシェルジュのみ表示・操作可能
- 削除時の成功メッセージ：「コンシェルジュを削除しました」
- 統計情報は日次更新
- 月間データは当月（表示月）のみを対象
