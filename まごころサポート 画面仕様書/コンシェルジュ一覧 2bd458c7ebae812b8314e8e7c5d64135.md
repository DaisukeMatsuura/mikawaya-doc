# コンシェルジュ一覧

ステータス: 確定
ユーザー種別: パートナー加盟店
概要: コンシェルジュの一覧表示
機能番号: M-003
画面種別: 一覧

---

## 🎯 画面の目的

パートナー加盟店の担当者が、自社に所属するコンシェルジュ（現場対応者）の情報を一覧表示し、詳細確認・編集・削除ができるようにする。

---

## 📊 表示データ項目

| 項目名 | データ型 | 備考 |
| --- | --- | --- |
| コンシェルジュ名 | String | 氏名 |
| 電話番号 | String | 連絡先 |
| メールアドレス | Email | 連絡先 |
| 担当案件数 | Number | 現在の進行中案件数 |
| ステータス | Enum | アクティブ / 非アクティブ |
| 登録日 | Date | YYYY-MM-DD形式 |
| 操作ボタン | Enum | 詳細 / 編集 / 削除 |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 新規登録ボタン | クリック | コンシェルジュ登録・編集モーダルを表示 |
| テーブル内の詳細ボタン | クリック | コンシェルジュ詳細画面へ遷移 |
| テーブル内の編集ボタン | クリック | コンシェルジュ登録・編集モーダルを表示 |
| テーブル内の削除ボタン | クリック | 削除確認ダイアログを表示 |
| フィルター（ステータス） | クリック | アクティブ/非アクティブで絞り込み |
| 検索ボックス | 入力 | コンシェルジュ名で検索（部分一致） |
| ソート（ヘッダー） | クリック | 該当カラムでソート（昇順/降順） |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データあり | 全コンシェルジュをテーブル表示 |
| データなし | 「コンシェルジュが登録されていません。」メッセージ |
| フィルター/検索後 | 該当するコンシェルジュのみ表示 |

---

## 🔐 権限

| ユーザー種別 | 閲覧 | 作成 | 編集 | 削除 |
| --- | --- | --- | --- | --- |
| 管理者 | ✕ | ✕ | ✕ | ✕ |
| オペレータ | ✕ | ✕ | ✕ | ✕ |
| パートナー | ◯（自店舗） | ◯ | ◯ | ◯ |
| コンシェルジュ | ✕ | ✕ | ✕ | ✕ |

---

## 🗑️ 削除ダイアログ

| 項目 | 内容 |
| --- | --- |
| タイトル | コンシェルジュ削除 |
| メッセージ | 「○○（コンシェルジュ名）を削除します。この操作は取り消せません。よろしいですか？」 |
| ボタン | キャンセル / 削除 |

---

## 📝 備考

- API: `GET /api/partner/concierges`
- PC対応（Responsive Web Design）
- パートナー加盟店は自店舗のコンシェルジュのみ表示・操作可能
- 削除時の成功メッセージ：「コンシェルジュを削除しました」
- テーブルのページネーション：1ページあたり20件表示
