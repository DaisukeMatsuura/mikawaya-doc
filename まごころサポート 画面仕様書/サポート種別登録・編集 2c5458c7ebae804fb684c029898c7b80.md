## 🎯 画面の目的

サポート種別マスタの新規登録および編集を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| サポート種別名 | String | ◯ |  |
| サポート種別名（ふりがな） | String | ◯ | ひらがなのみ |
| 有効/無効 | Enum | ◯ | 有効 / 無効、デフォルト：有効 |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| サポート種別名入力欄 | 入力 | 必須バリデーション |
| サポート種別名（ふりがな）入力欄 | 入力 | ひらがなのみ許可 |
| 有効/無効セレクト | 選択 | 有効 / 無効 |
| キャンセルボタン | クリック | モーダルを閉じる（入力内容は破棄） |
| 登録ボタン | クリック | 新規登録処理を実行 |
| 更新ボタン（編集時） | クリック | 更新処理を実行 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 新規登録モード | フォーム空欄、ボタン「登録」 |
| 編集モード | 既存データを表示、ボタン「更新」 |
| バリデーションエラー | 該当項目を赤枠 + エラーメッセージ |
| 処理中 | ボタンをローディング表示 |
| 登録/更新完了 | モーダルを閉じる、一覧を更新、トースト通知 |

| --- | --- | --- |

---

## 📝 備考

- API: `POST /api/admin/master/support-types`（新規登録）
- API: `PUT /api/admin/master/support-types/{id}`（編集）
- 新規登録時のデフォルトは「有効」
- 重複チェック：同名のサポート種別は登録不可
