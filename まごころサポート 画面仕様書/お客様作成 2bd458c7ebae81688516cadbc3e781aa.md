## 🎯 画面の目的

新規のお客様（顧客）情報を登録する。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 案件No | String | - | 自動表示（案件から登録した場合） |
| 姓 | String | ◯ |  |
| 名 | String | ◯ |  |
| 姓(ふりがな) | String | ◯ | ひらがなのみ |
| 名(ふりがな) | String | ◯ | ひらがなのみ |
| 担当店舗 | Enum | ◯ | セレクト（店舗マスタから選択） |
| 担当コンシェルジュ | Enum | - | セレクト（コンシェルジュマスタから選択） |
| 住所 | String | - |  |
| 電話番号 | String | - |  |
| 地域包括支援センター・施設など | Boolean | - | トグル、ツールチップあり |
| 登録経路 | Enum | - | セレクト |
| 紹介者コード | String | - |  |
| コールセンター専用メモ | String | - | テキストエリア |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 姓・名入力欄 | 入力 | 必須バリデーション |
| ふりがな入力欄 | 入力 | ひらがなのみ許可 |
| 担当店舗セレクト | 選択 | 店舗を選択 |
| 担当コンシェルジュセレクト | 選択 | 選択した店舗に所属するコンシェルジュのみ表示 |
| 住所入力欄 | 入力 | 自由入力 |
| 電話番号入力欄 | 入力 | 数字・ハイフンのみ許可 |
| 地域包括支援センタートグル | クリック | ON/OFF切り替え |
| 登録経路セレクト | 選択 | 登録経路を選択 |
| 紹介者コード入力欄 | 入力 | 自由入力 |
| コールセンター専用メモ | 入力 | テキストエリア |
| キャンセルボタン | クリック | モーダルを閉じる（入力内容は破棄） |
| 登録ボタン | クリック | お客様を登録 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 新規登録モード | フォーム空欄 |
| 案件からの登録 | 案件Noを表示 |
| バリデーションエラー | 該当項目を赤枠 + エラーメッセージ |
| 処理中 | ボタンをローディング表示 |
| 登録完了 | モーダルを閉じる、一覧を更新、トースト通知「お客様を登録しました」 |

## 📝 備考

- API: `POST /api/operator/customers`
- 会員番号は登録時に自動採番
- 担当店舗を選択すると、その店舗に所属するコンシェルジュのみが担当コンシェルジュに表示される
- パートナー・コンシェルジュが登録する場合、担当店舗は自店舗で固定
- 詳細情報（住所詳細、関係者情報等）は登録後にお客様詳細画面で追加
