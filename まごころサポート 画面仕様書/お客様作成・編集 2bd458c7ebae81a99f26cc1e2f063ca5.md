## 🎯 画面の目的

パートナー加盟店が新規のお客様（顧客）情報を登録する。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 案件No | String | - | 自動表示（案件から登録した場合） |
| 顧客ステータス | Enum | ◯ | 本登録 等 |
| 担当店舗 | Enum | ◯ | 自店舗で固定 |
| 登録経路 | Enum | - | セレクト |
| お客様担当コンシェルジュ | Enum | - | セレクト（自店舗のコンシェルジュのみ） |
| 基本支払方法 | Enum | - | セレクト |
| 地域包括支援センター・施設など | Boolean | - | トグル、ツールチップあり |
| 基本代金回収方法 | Enum | - | 店舗が後日回収予定 等 |
| ご逝去 | Boolean | - | チェックボックス |
| 関係性 | Enum | - | 知り合い 等 |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 顧客ステータスセレクト | 選択 | ステータスを選択 |
| 担当店舗 | - | 自店舗で固定（変更不可） |
| 登録経路セレクト | 選択 | 登録経路を選択 |
| お客様担当コンシェルジュセレクト | 選択 | 自店舗のコンシェルジュから選択 |
| 基本支払方法セレクト | 選択 | 支払方法を選択 |
| 地域包括支援センタートグル | クリック | ON/OFF切り替え |
| 基本代金回収方法セレクト | 選択 | 回収方法を選択 |
| ご逝去チェックボックス | クリック | チェックON/OFF |
| 関係性セレクト | 選択 | 関係性を選択 |
| キャンセルボタン | クリック | モーダルを閉じる（入力内容は破棄） |
| 保存ボタン | クリック | お客様を登録 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 新規登録モード | フォーム空欄（担当店舗は自店舗で固定） |
| 案件からの登録 | 案件Noを表示 |
| バリデーションエラー | 該当項目を赤枠 + エラーメッセージ |
| 処理中 | ボタンをローディング表示 |
| 登録完了 | モーダルを閉じる、一覧を更新、トースト通知「お客様を登録しました」 |

## 📝 備考

- API: `POST /api/partner/customers`
- 会員番号は登録時に自動採番
- 担当店舗は自店舗で固定（変更不可）
- お客様担当コンシェルジュは自店舗に所属するコンシェルジュのみ選択可能
- 詳細情報（住所、関係者情報等）は登録後にお客様詳細画面で追加

## オペレーター版との違い

| 項目 | オペレーター | パートナー |
| --- | --- | --- |
| 担当店舗 | 全店舗から選択可 | 自店舗で固定 |
| 顧客ステータス | なし | あり |
| 基本支払方法 | なし | あり |
| 基本代金回収方法 | なし | あり |
