## 🎯 画面の目的

パートナー加盟店の詳細情報を表示し、編集を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 店舗コード | String | ◯ | 編集不可（自動採番） |
| 店舗名 | String | ◯ |  |
| 都道府県 | String | ◯ |  |
| 対応エリア | String | ◯ |  |
| 責任者名 | String | ◯ |  |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 編集ボタン（右上） | クリック | パートナー加盟店編集モーダルを表示 |

---

## 🔄 状態パターン

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | ローディングスピナー |
| データ表示 | 各項目を表示 |

## 📝 備考

- API: `GET /api/admin/partners/{id}`
- 店舗コードは編集不可
- 編集は右上の「編集」ボタンからモーダルで行う
