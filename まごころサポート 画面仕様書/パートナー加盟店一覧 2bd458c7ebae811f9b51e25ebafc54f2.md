## 🎯 画面の目的

パートナー加盟店の一覧を表示し、店舗情報の管理を行う。

---

## 📊 表示データ項目

| 項目名 | データ型 | 必須 | 備考 |
| --- | --- | --- | --- |
| 店舗コード | String | ◯ | ソート可能 |
| 店舗名 | String | ◯ | ソート可能 |
| 都道府県 | String | ◯ | ソート可能 |
| 対応エリア | String | ◯ | ソート可能 |
| 責任者 | String | ◯ | ソート可能 |
| ステータス | Enum | ◯ | アクティブ / 非アクティブ |

---

## ⚙️ 機能・操作

| 要素 | 操作 | 動作 |
| --- | --- | --- |
| 店舗コードカラムヘッダー | クリック | 店舗コードで昇順/降順ソート |
| 店舗名カラムヘッダー | クリック | 店舗名で昇順/降順ソート |
| 都道府県カラムヘッダー | クリック | 都道府県で昇順/降順ソート |
| 対応エリアカラムヘッダー | クリック | 対応エリアで昇順/降順ソート |
| 責任者カラムヘッダー | クリック | 責任者で昇順/降順ソート |
| 行 | クリック | パートナー加盟店詳細画面へ遷移 |
| 三点メニュー（︙） | クリック | メニュー表示（編集、削除等） |
| パートナー加盟店登録ボタン | クリック | パートナー加盟店登録モーダルを表示 |
| ページネーション | クリック | ページ切り替え |

## 三点メニュー

| 項目 | 動作 |
| --- | --- |

※非アクティブの店舗には「アクティブにする」を表示

## 📝 備考

- API: `GET /api/admin/partners`
- 店舗コードは自動採番（YYYYMMDD + 連番）
    - 採番方法は要検討
- 削除は論理削除（ステータスを「非アクティブ」に変更）
- 非アクティブの店舗は一覧にデフォルト非表示（フィルターで表示切替可能）
- 非アクティブの店舗は案件割り当て不可
- 過去の案件データとの紐づきは維持される
